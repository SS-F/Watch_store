# Common Watch Movement Types

| Movement Type | Description | Famous Companies / Watches |
|---|---|---|
| Mechanical (Manual) | Hand-wound movement powered by winding the crown | Patek Philippe, Omega Speedmaster Professional |
| Automatic (Self-Winding) | Mechanical movement powered by wrist motion | Rolex Oyster Perpetual, Seiko 5 |
| Quartz | Battery-powered movement with high accuracy | Casio, Citizen, Seiko Quartz |
| Solar Quartz | Quartz movement powered by solar energy | Citizen Eco-Drive, Seiko Solar |
| Kinetic | Combines automatic rotor with quartz technology | Seiko Kinetic |
| Spring Drive | Hybrid of mechanical and quartz regulation | Grand Seiko Spring Drive |
| Digital | Electronic display using LCD/LED | Casio G-Shock, Timex Digital |
| Smartwatch Movement | Software-driven digital operating system | Apple Watch, Samsung Galaxy Watch |
| Tourbillon | Advanced mechanical complication improving accuracy | Breguet Tourbillon, Audemars Piguet |
| Hybrid Movement | Combines traditional analog with smart features | Withings, Garmin Hybrid Watches |
